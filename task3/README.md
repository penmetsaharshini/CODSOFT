# 🧮 CALCULATOR

A sleek, responsive, and fully functional calculator web application built using **HTML5, CSS3 (Grid & Flexbox), and JavaScript**. This project was developed as part of my **Web Development Internship at CodSoft (Level 1, Task 3)**.

---

## ✨ Key Features
- **Modern UI/UX:** Features a professional Dark Theme with **Glassmorphism** effects and smooth hover animations.
- **Advanced Operations:** Beyond basic math, it supports percentages (%), decimal points, and a dedicated **DEL** (Backspace) key for error correction.
- **Grid Layout:** Optimized button alignment using the **CSS Grid System** for a balanced and clean interface.
- **Smart Logic:** - Prevents multiple zeros at the start.
  - Automatically resets the screen after a calculation is completed and a new number is pressed.
  - Error handling for invalid expressions (e.g., dividing by zero).
- **Responsive Design:** Works flawlessly on both mobile devices and desktops.

## 🛠️ Tech Stack Used
* **HTML5:** For the semantic structure of the calculator.
* **CSS3:** * **CSS Grid:** For building the button layout.
    * **Flexbox:** For centering the calculator on the screen.
    * **Backdrop-filter:** For the frosted glass effect.
* **JavaScript (ES6+):** * **DOM Manipulation:** To update the display in real-time.
    * **Event Handling:** For button clicks.
    * **Logic:** Implementing mathematical calculations and input validation.
