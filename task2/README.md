# 🌿 Verdant Aura — Botanical Boutique Landing Page

---

## 🍃 Project Overview
This project is a high-end, eco-minimalist landing page for **"Verdant Aura,"** a premium botanical boutique. Developed as part of my **CodSoft Internship**, the goal was to create an "expensive-looking" UI that is both functional and visually stunning.

> "Bringing nature to your digital doorstep with modern design."

## ✨ Key Features
| Feature | Description |
| :--- | :--- |
| 🌒 **Dark Mode** | A sleek, professional dark aesthetic with deep green accents. |
| 📱 **Responsive** | Fully optimized for Mobile, Tablet, and Desktop screens. |
| 🛍️ **Collections** | Elegant grid layouts showcasing exotic plants and boutique items. |
| 🧼 **Glassmorphism** | Modern navigation bar with frosted glass blur effects. |
| ✍️ **Typography** | Premium font pairing (Playfair Display & Plus Jakarta Sans). |

## 🛠️ Tech Stack
* 🧱 **HTML5** — Semantic structure for SEO and accessibility.
* 🎨 **CSS3** — Custom styling using **Flexbox** and **CSS Grid**.
* 🖋️ **Google Fonts** — Professional typography for a luxury feel.
* 📦 **Lucide Icons** — Minimalist icons for a clean user interface.
